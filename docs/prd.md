## 1. 상품 개요

### 1.1 상품명

**키워드 검색 광고 (Keyword Search Ads)** - 내부 코드명: **KSA**

### 1.2 상품 목적

검색 사용자가 입력한 키워드와 광고주가 등록한 키워드의 매칭을 기반으로, 검색 결과 페이지(SERP) 상단 및 하단에 텍스트형 광고를 노출하는 클릭 과금(CPC) 광고 상품이다. 광고주가 직접 캠페인을 생성하고 운영할 수 있는 셀프서브(Self-Serve) 플랫폼을 제공하여, 중소 광고주부터 대형 광고주까지 진입 장벽 없이 사용할 수 있도록 한다.

### 1.3 타겟 광고주

| 세그먼트 | 설명 | 월 예상 예산 | 핵심 니즈 |
|---------|------|------------|----------|
| **중소상공인(SMB)** | 자영업자, 소규모 온라인 셀러 | 30만~300만 원 | 간편한 셀프서브, 낮은 최소 입찰가, 명확한 성과 리포팅 |
| **온라인 커머스** | 쇼핑몰, D2C 브랜드 | 300만~3,000만 원 | ROAS 최적화, 상품 피드 연동, 전환 추적 |
| **대형 광고주/에이전시** | 브랜드 광고주, 미디어렙 | 3,000만 원 이상 | 대량 키워드 관리, API 연동, 커스텀 리포팅 |
| **로컬 비즈니스** | 음식점, 학원, 병원 등 지역 업종 | 10만~100만 원 | 지역 타겟팅, 전화/방문 전환 추적 |
| **앱/게임** | 앱 퍼블리셔, 게임사 | 500만~5,000만 원 | 앱 설치 전환, 딥링크 지원 |

### 1.4 핵심 가치 제안 (Value Proposition)

1. **설명 가능한 AI 자동입찰**: 입찰 조정 사유를 투명하게 제공하여 광고주가 AI 의사결정을 이해하고 신뢰할 수 있다. 경쟁사의 블랙박스 자동입찰과 차별화한다.
2. **퍼스트파티 데이터 기반 타겟팅**: 서드파티 쿠키 퇴출 환경에서 자사 플랫폼 로그인 데이터와 광고주 CRM 데이터를 결합하여 정밀한 오디언스 타겟팅을 제공한다.
3. **중소 광고주 친화적 셀프서브 UX**: 5분 내 캠페인 생성 완료가 가능한 간결한 UI, AI 키워드 추천, 자동 소재 생성 기능을 제공한다.
4. **경쟁력 있는 CPC**: 네이버 파워링크 대비 20~30% 낮은 평균 CPC를 목표로, 최소 입찰가 50원으로 진입 장벽을 낮춘다.

## 2. 핵심 기능

### 2.1 캠페인 관리

#### 2.1.1 캠페인 계층 구조

```
계정 (Account)
 └── 캠페인 (Campaign)
      └── 광고그룹 (Ad Group)
           ├── 키워드 (Keywords)
           └── 광고 소재 (Ads)
```

#### 2.1.2 캠페인 생성/수정/삭제

| 항목 | 상세 |
|------|------|
| **캠페인 유형** | 검색 광고 (Phase 1), 쇼핑 검색 광고 (Phase 2) |
| **캠페인명** | 최대 100자, 중복 허용 |
| **일일 예산** | 최소 1,000원, 1원 단위 설정 |
| **월 예산 한도** | 일일 예산 x 30.4 (월 평균 일수) 자동 계산, 수동 설정 가능 |
| **노출 지역** | 전국 / 시/도 단위 / 시/군/구 단위 선택 |
| **노출 디바이스** | PC, 모바일, 전체 (디바이스별 입찰 가중치 설정 가능: -90% ~ +900%) |
| **노출 시간대** | 요일별, 시간대별 스케줄 설정 (1시간 단위) |
| **캠페인 삭제** | 소프트 삭제 (30일 보관 후 완전 삭제), 삭제 시 하위 광고그룹/키워드/소재 일괄 중지 |

#### 2.1.3 캠페인 상태 관리

| 상태 | 설명 |
|------|------|
| **활성 (Active)** | 정상 노출 중 |
| **일시중지 (Paused)** | 광고주가 수동 중지, 재개 가능 |
| **예산소진 (Budget Exhausted)** | 일일/월 예산 소진으로 자동 중지, 익일/익월 자동 재개 |
| **기간종료 (Ended)** | 설정된 종료일 도달 |
| **검토중 (Under Review)** | 소재 심사 진행 중 |
| **제한됨 (Limited)** | 정책 위반 또는 잔액 부족으로 노출 제한 |
| **삭제됨 (Deleted)** | 소프트 삭제 상태 |

#### 2.1.4 일정 설정

- **시작일/종료일**: 시작일 필수, 종료일 선택 (미설정 시 무기한)
- **시작일 최소값**: 캠페인 생성일 당일
- **시간대**: KST(UTC+9) 기준

### 2.2 키워드 관리

#### 2.2.1 키워드 등록

| 항목 | 상세 |
|------|------|
| **키워드 길이** | 최소 1자, 최대 50자 (공백 포함) |
| **광고그룹당 키워드 수** | 최대 5,000개 |
| **캠페인당 키워드 수** | 최대 50,000개 |
| **계정당 키워드 수** | 최대 500,000개 |
| **허용 문자** | 한글, 영문, 숫자, 공백 (특수문자 불가, 단 `-`, `_` 허용) |
| **키워드 중복** | 동일 광고그룹 내 동일 매치 타입 중복 불가 |

#### 2.2.2 매치 타입 (Match Type)

| 매치 타입 | 기호 | 설명 | 예시 (등록: "남성 운동화") |
|----------|------|------|--------------------------|
| **정확 일치 (Exact)** | `[키워드]` | 검색어가 키워드와 정확히 일치할 때 노출. 어순 변경, 조사 변형은 허용. | "남성 운동화", "남성운동화" |
| **구문 일치 (Phrase)** | `"키워드"` | 검색어에 키워드 구문이 포함될 때 노출. 앞뒤 추가 단어 허용. | "저렴한 남성 운동화 추천" |
| **확장 일치 (Broad)** | 키워드 | 키워드와 의미적으로 관련된 검색어에 노출. 동의어, 유사어 포함. | "남자 스니커즈", "신발 남성용" |
| **AI 확장 (AI Broad)** | `+키워드` | 머신러닝 기반으로 전환 가능성이 높은 관련 검색어로 자동 확장. | "조깅화 남자", "러닝화 추천 남성" |

#### 2.2.3 제외 키워드 (Negative Keywords)

- **적용 레벨**: 캠페인 단위, 광고그룹 단위
- **매치 타입**: 정확 일치, 구문 일치 지원
- **제외 키워드 수**: 광고그룹당 최대 1,000개, 캠페인당 최대 10,000개
- **공유 제외 키워드 목록**: 계정 단위로 제외 키워드 목록 생성 후 여러 캠페인에 일괄 적용 가능

#### 2.2.4 키워드 도구

| 기능 | 설명 |
|------|------|
| **키워드 추천** | 업종/URL 입력 시 AI 기반 키워드 후보 추천 (검색량, 경쟁도, 예상 CPC 포함) |
| **검색량 조회** | 키워드별 월간 검색량, 트렌드 그래프 제공 |
| **예상 성과 시뮬레이터** | 입찰가/예산 설정 시 예상 노출수, 클릭수, 비용 시뮬레이션 |
| **키워드 진단** | 등록 키워드의 품질지수, 입찰 경쟁 상황, 개선 제안 제공 |

### 2.3 광고 소재 관리

#### 2.3.1 텍스트 광고 구성 요소

| 요소 | 규격 | 필수 여부 |
|------|------|----------|
| **제목 1** | 최대 30자 | 필수 |

| **최종 URL** | 랜딩 페이지 URL | 필수 |

#### 2.3.2 확장 소재 (Ad Extensions)

| 확장 유형 | 설명 | Phase |
|----------|------|-------|
| **사이트링크** | 추가 링크 2~6개 표시 (제목 25자, 설명 35자 x 2줄) | Phase 1 |
| **전화번호** | 클릭 투 콜 기능, 전화 전환 추적 | Phase 1 |
| **위치정보** | 매장 주소, 지도 연동 | Phase 1 |
| **콜아웃** | 짧은 홍보 문구 (각 25자, 최대 10개) | Phase 1 |
| **가격** | 상품/서비스 가격 목록 | Phase 2 |
| **프로모션** | 할인/이벤트 정보 (기간 설정 가능) | Phase 2 |
| **앱 설치** | 앱 스토어 다운로드 링크 | Phase 2 |
| **구조화 스니펫** | 카테고리별 상세 정보 나열 | Phase 2 |

#### 2.3.3 AI 소재 생성 (Phase 2)

- 광고주의 랜딩 페이지 URL과 키워드를 입력하면 AI가 제목/설명 후보를 자동 생성
- A/B 테스트 자동 운영: 최대 15개 제목, 4개 설명 조합 중 성과 최적 조합 자동 선별
- 광고주 최종 승인 후 적용

#### 2.3.4 심사 프로세스

```
소재 등록 → 자동 심사 (실시간) → [통과] → 노출 시작
                                → [보류] → 수동 심사 (영업일 기준 1일 이내)
                                            → [승인] → 노출 시작
                                            → [반려] → 반려 사유 안내 → 수정 후 재심사
```

| 심사 항목 | 상세 |
|----------|------|
| **자동 심사** | 금칙어 필터, URL 유효성, 문자 규격, 금지 업종 키워드 탐지 |
| **수동 심사** | 과장 광고, 비교 광고 적정성, 법적 표기 사항, 랜딩 페이지 내용 일치 여부 |
| **심사 SLA** | 자동 심사: 등록 후 10초 이내, 수동 심사: 영업일 기준 24시간 이내 |
| **재심사** | 소재 수정 시 자동 재심사 트리거, 기존 승인 소재는 수정 전까지 유지 |

### 2.4 리포팅

#### 2.4.1 핵심 지표 (Metrics)

| 지표 | 정의 | 계산식 |
|------|------|--------|
| **노출수 (Impressions)** | 광고가 검색 결과에 표시된 횟수 | 카운트 |
| **클릭수 (Clicks)** | 광고 클릭 횟수 (중복 클릭 필터링 적용) | 유효 클릭 카운트 |
| **CTR (클릭률)** | 노출 대비 클릭 비율 | 클릭수 / 노출수 x 100 |
| **CPC (클릭당 비용)** | 클릭 1회당 평균 비용 | 총 비용 / 클릭수 |
| **총 비용 (Cost)** | 과금된 총 광고비 | 유효 클릭별 과금액 합계 |
| **전환수 (Conversions)** | 설정된 전환 액션 완료 횟수 | 전환 추적 스크립트 기반 |
| **전환율 (CVR)** | 클릭 대비 전환 비율 | 전환수 / 클릭수 x 100 |
| **전환당 비용 (CPA)** | 전환 1건당 평균 비용 | 총 비용 / 전환수 |
| **ROAS** | 광고비 대비 매출 비율 | 전환 매출 / 총 비용 x 100 |
| **평균 게재 순위 (Avg. Position)** | 광고가 노출된 평균 순위 | 순위 가중 평균 |
| **노출 점유율 (Impression Share)** | 가능한 총 노출 대비 실제 노출 비율 | 실제 노출 / 적격 노출 x 100 |

#### 2.4.3 리포트 기능

- **기간 설정**: 오늘, 어제, 최근 7일, 최근 30일, 사용자 정의 기간 (최대 90일)
- **데이터 갱신 주기**: 실시간 (최대 3시간 지연), 일간 집계는 익일 09:00 확정
- **다운로드**: CSV, Excel 형식 지원
- **자동 리포트**: 일간/주간/월간 자동 발송 (이메일)
- **대시보드**: 핵심 KPI 요약, 추세 그래프, 이상 탐지 알림

## 3. 과금 모델

### 3.1 과금 방식: CPC (Cost Per Click)

광고주는 광고 클릭이 발생한 경우에만 비용을 지불한다. 노출만으로는 과금되지 않는다.

| 항목 | 상세 |
|------|------|
| **과금 시점** | 유효 클릭 발생 시 즉시 차감 |
| **유효 클릭 판정** | 무효 클릭 필터링 시스템 적용 (중복 클릭, 봇 클릭, 경쟁사 악성 클릭 탐지 및 제외) |
| **무효 클릭 환급** | 실시간 필터링 미탐지분은 월 1회 사후 정산하여 크레딧으로 환급 |
| **과금 단위** | 1원 단위 |

### 3.2 입찰 구조: 일반화된 2차가 경매 (Generalized Second Price, GSP)

#### 3.2.1 경매 방식

광고주가 설정한 최대 입찰가(Max CPC)보다 실제 과금액(Actual CPC)이 낮거나 같다. 실제 과금액은 바로 아래 순위 광고의 Ad Rank를 이기기 위해 필요한 최소 금액으로 결정된다.

**실제 과금 공식:**

```
Actual CPC = (아래 순위 Ad Rank / 내 품질지수) + 1원
```

- 최하위 노출 광고의 경우: `Actual CPC = 최소 입찰가`
- Actual CPC는 광고주의 Max CPC를 초과하지 않음

#### 3.2.2 입찰 구조 예시

| 광고주 | Max CPC | 품질지수 | Ad Rank | 게재 순위 | Actual CPC |
|--------|---------|---------|---------|----------|-----------|
| A | 500원 | 8 | 4,000 | 1위 | (3,150 / 8) + 1 = **395원** |
| B | 450원 | 7 | 3,150 | 2위 | (2,400 / 7) + 1 = **344원** |
| C | 400원 | 6 | 2,400 | 3위 | (1,750 / 6) + 1 = **293원** |
| D | 350원 | 5 | 1,750 | 4위 | **50원** (최소 입찰가) |

### 3.3 입찰 옵션

| 입찰 전략 | 설명 | Phase |
|----------|------|-------|
| **수동 입찰 (Manual CPC)** | 광고주가 키워드별 Max CPC를 직접 설정 | Phase 1 |
| **자동 입찰 - 클릭 최대화 (Maximize Clicks)** | 일일 예산 내에서 클릭수를 최대화하도록 AI가 입찰가 자동 조정 | Phase 1 |
| **자동 입찰 - 목표 CPA (Target CPA)** | 목표 전환당 비용을 설정하면 AI가 입찰가 최적화 | Phase 2 |
| **자동 입찰 - 목표 ROAS (Target ROAS)** | 목표 ROAS를 설정하면 AI가 입찰가 최적화 | Phase 2 |

**설명 가능한 AI 자동입찰 (차별화 기능):**

자동입찰 적용 시 다음 정보를 광고주에게 투명하게 제공한다.

- 입찰 조정 사유: "검색 시간대(오후 2시)와 디바이스(모바일)에서 전환율이 평균 대비 40% 높아 입찰가를 300원 → 420원으로 상향 조정했습니다."
- 주요 입찰 변동 이력 로그
- 입찰 전략 성과 요약 (주간 리포트)

## 4. 광고 정책

### 4.1 품질지수 (Quality Score)

품질지수는 1~10점 척도로 산정되며, 광고 순위와 실제 CPC에 직접 영향을 미친다.

#### 4.1.1 구성 요소

| 구성 요소 | 비중 | 평가 기준 |
|----------|------|----------|
| **예상 CTR (Expected CTR)** | 40% | 해당 키워드에서의 예상 클릭률. 과거 CTR, 키워드-소재 관련성, 게재 위치 보정. 평가: 평균 이상 / 평균 / 평균 이하 |
| **소재 관련성 (Ad Relevance)** | 30% | 키워드와 광고 소재(제목, 설명)의 의미적 일치도. NLP 기반 분석. 평가: 평균 이상 / 평균 / 평균 이하 |
| **랜딩페이지 경험 (Landing Page Experience)** | 30% | 페이지 로딩 속도, 모바일 최적화, 키워드-콘텐츠 관련성, 사용자 체류 시간. 평가: 평균 이상 / 평균 / 평균 이하 |

#### 4.1.2 품질지수 산정 세부 규칙

- **신규 키워드**: 초기 품질지수 6점 부여 (중립), 데이터 축적 후 재산정
- **갱신 주기**: 일 1회 (매일 04:00 KST 기준 재계산)
- **데이터 최소 요건**: 노출 1,000회 이상 시 통계적으로 유의미한 품질지수 산출
- **계정 이력 반영**: 계정 전체의 과거 CTR 이력이 신규 키워드 초기 품질지수에 보정 영향

### 4.2 광고 순위 결정 공식

#### 4.2.1 Ad Rank 공식

```
Ad Rank = Max CPC × Quality Score × Ad Extension Factor
```

| 요소 | 설명 |
|------|------|
| **Max CPC** | 광고주가 설정한 최대 클릭당 입찰가 |
| **Quality Score** | 품질지수 (1~10) |
| **Ad Extension Factor** | 확장 소재 적용 보너스 (1.0 ~ 1.2). 확장 소재가 많고 성과가 좋을수록 최대 20% 가산 |

#### 4.2.2 노출 기준

- **최소 Ad Rank 기준(Ad Rank Threshold)**: 검색어별로 동적 설정. 경쟁이 없는 검색어에서도 품질이 낮은 광고는 노출 차단.
- **상단 노출 기준**: Ad Rank가 상단 노출 Threshold 이상인 광고만 검색 결과 상단(최대 4개)에 노출
- **하단 노출**: 상단 기준 미달이나 최소 기준 충족 광고는 검색 결과 하단(최대 3개)에 노출
- **동일 Ad Rank 시**: 먼저 등록된 광고에 우선순위 부여

#### 4.2.3 노출 슬롯

| 위치 | 슬롯 수 | 설명 |
|------|--------|------|
| 검색 결과 상단 | 최대 4개 | "광고" 라벨 표시, 프리미엄 영역 |
| 검색 결과 하단 | 최대 3개 | "광고" 라벨 표시 |

### 4.3 광고 심사 기준

#### 4.3.1 금지 업종

다음 업종은 키워드 검색 광고 집행이 전면 금지된다.

| 카테고리 | 상세 |
|---------|------|
| **불법 상품/서비스** | 마약, 불법 도박, 불법 무기, 위조품, 해킹 도구 |
| **성인 콘텐츠** | 음란물, 성인 용품 (일부 허용 업종은 별도 심사) |
| **금융 사기** | 불법 대출, 유사 수신, 미등록 금융업 |
| **건강 위해** | 미승인 의약품, 허위 건강 효능 주장 |
| **법적 규제 업종** | 담배 광고, 주류 (방송광고 규제 준용), 총포/화약류 |

#### 4.3.2 소재 가이드라인

| 항목 | 기준 |
|------|------|
| **허위/과장** | 객관적으로 입증 불가한 "최고", "1위", "최저가" 등 최상급 표현 금지 (공인 기관 인증 시 예외) |
| **비교 광고** | 경쟁사 직접 언급 금지, 객관적 데이터 기반 간접 비교는 허용 |
| **가격 표기** | 가격 명시 시 부가세 포함 최종가 표기 필수 |
| **특수 기호** | 과도한 특수문자 반복 사용 금지 (예: ★★★, !!!) |
| **연락처** | 소재 본문에 전화번호 직접 기재 금지 (전화번호 확장 소재 사용) |
| **URL 정책** | 표시 URL과 최종 URL의 도메인 일치 필수, 리다이렉트 금지 |
| **법적 표기** | 의료/법률/금융 업종은 면허번호, 등록번호 표기 필수 |
| **문법/맞춤법** | 의도적 오탈자, 자음/모음 분리 표기 금지 (예: "ㅋㅋㅋ", "대ㅂㅏㄱ") |

#### 4.3.3 키워드 정책

| 항목 | 기준 |
|------|------|
| **상표권** | 타사 상표 키워드 입찰은 허용하되, 소재 본문에 타사 상표 사용은 금지 |
| **금칙 키워드** | 불법/유해 검색어와 매칭되는 키워드 등록 차단 |
| **키워드-소재 일관성** | 키워드와 무관한 소재 집행 시 품질지수 하락 및 반려 조치 |

#### 4.3.4 정책 위반 제재

| 위반 수준 | 조치 |
|----------|------|
| **경미** | 소재 반려, 수정 요청 |
| **반복 위반 (3회 이상)** | 계정 경고, 해당 캠페인 7일 정지 |
| **중대 위반** | 소재 즉시 삭제, 계정 30일 정지 |
| **악의적/불법** | 계정 영구 정지, 잔액 동결, 법적 조치 |

## 부록

### A. 용어 정의

| 용어 | 정의 |
|------|------|
| **CPC** | Cost Per Click. 클릭당 비용. |
| **CTR** | Click-Through Rate. 노출 대비 클릭 비율. |
| **CVR** | Conversion Rate. 클릭 대비 전환 비율. |
| **ROAS** | Return On Ad Spend. 광고비 대비 매출 비율. |
| **CPA** | Cost Per Acquisition. 전환당 비용. |
| **GSP** | Generalized Second Price. 일반화된 2차가 경매. |
| **Ad Rank** | 광고 순위 결정 점수. 입찰가 x 품질지수 x 확장 소재 보너스. |
| **Quality Score** | 품질지수. 광고의 품질을 1~10으로 수치화한 점수. |
| **SERP** | Search Engine Results Page. 검색 결과 페이지. |
| **PMF** | Product-Market Fit. 제품-시장 적합성. |
