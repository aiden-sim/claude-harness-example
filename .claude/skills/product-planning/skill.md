# Product Planning — 키워드 광고 상품 기획서 작성

## 워크플로우

### Step 1: 시장 분석 결과 검토
- `docs/market-analysis.md` 파일을 읽고 핵심 인사이트 파악
- 차별화 포인트와 기회 영역 확인

### Step 2: 상품 개요 정의
- 상품명, 상품 목적, 타겟 광고주 정의
- 핵심 가치 제안(Value Proposition) 수립
- 경쟁 우위 포인트 명시

### Step 3: 핵심 기능 설계
- 캠페인 관리: 생성/수정/삭제, 상태 관리, 일정 설정
- 키워드 관리: 키워드 등록, 매치 타입(정확/구문/확장), 제외 키워드
- 광고 소재 관리: 텍스트 광고, 확장 소재, 심사 프로세스
- 리포팅: 노출수, 클릭수, CTR, CPC, 전환율, ROAS

### Step 4: 과금 모델 설계
- CPC 과금 체계 상세 정의
- 입찰 구조 설계 (2차가 경매 / Generalized Second Price)
- 최소 입찰가, 일일 예산, 월 예산 한도 정책
- 크레딧/충전 방식 정의

### Step 5: 광고 정책 수립
- 품질지수 구성 요소 정의 (예상 CTR, 소재 관련성, 랜딩페이지 경험)
- 광고 순위 결정 공식: Ad Rank = 입찰가 × 품질지수
- 광고 심사 기준 (금지 업종, 소재 가이드라인)

### Step 6: MVP 범위 정의
- Phase 1 (MVP) 기능 목록 확정
- Phase 2 확장 기능 로드맵
- 비기능 요구사항 (성능, 확장성, SLA)

### Step 7: 산출물 작성
- `docs/prd.md` 파일로 기획서 작성
- 출력 형식은 product-planner 에이전트의 출력 형식을 따른다
